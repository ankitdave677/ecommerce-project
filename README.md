# Database Schema Design

## Products Schema (MongoDB)
- `name: String`
- `description: String`
- `price: Number`
- `category: String`
- `stock: Number`
- `imageUrl: String`

## Users Schema (MongoDB)
- `username: String`
- `password: String`
- `email: String`

## Orders Schema (MongoDB)
- `order_id: String`
- `user_id: String`
- `product_id: String`
- `quantity: Number`
- `order_date: Date`
